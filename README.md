# MarineEng Academy

**Free knowledge hub for aspiring marine engineers.**

Live site → [https://your-username.github.io/marineeng-academy](https://your-username.github.io/marineeng-academy)

## What's inside

24 topic modules covering the full spectrum of marine engineering — from Boiler Systems to LNG Vessel Operations, with a dedicated licensing exam prep section (3AE/2AE/1AE oral exam past questions).

| Category | Topics |
|---|---|
| Machinery | Boiler, Diesel Generator, Main Engine, Main Turbine, Turbo Generator, Emergency Generator, Purifier, Governor |
| Electrical | Electric Motors & Alignment, Electrical Systems, Switchboard |
| Safety | LSA & Fire Fighting, Emergency Drills |
| Auxiliary | Pumps, Refrigeration, AHU/HVAC, Steering, Valves & Piping, CAPAC, MGPS |
| Engineering | Control Engineering & PID, Fuel Oil Properties |
| Certification | Marine Engineer Licensing Exam, Maritime Conventions |
| Specialist | LNG Vessel Operations |
| Practical | Tools, Parts & Materials |

## Source

Materials compiled from real onboard documents used aboard NYK Line and MOL vessels. Original files are in Japanese; English summaries are provided for each topic.

## Deploy to GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages → Source → Deploy from branch → `main` / root**
3. Your site will be live at `https://your-username.github.io/marineeng-academy`
